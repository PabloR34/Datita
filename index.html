Comandos de Git y Github


$ git init
$ git add README.md
$ git status (muestra los archivos que han sido modificados)
$ git add . (agrega cambios al repositorio local)
$ git commit -m "detallar, dar informacion de lo que se esta modificando"
$ git config --global user.email "Colocar correo electronico"
$ git confing --global user.name "colocar nombre"
$ git branch -M main (con esto indicamos que subimos la modificacion a la rama principal)
$ git remote add origin https://github.com/"......."(colocar la ruta que proporciona github para cada repositorio
Indicamos que subimos el cambio al repositorio remoto $ git push -u origin main (main = o como se llame la rama principal, si es una rama el nombre de la rama)
indicamos que vamos a hacer efectiva la sincronizacion del repositorio local con el remoto

#TAG traer ultimos cambios de la master branch origin/main
$ git fetch --all
$ git add --all
$ git stash
$ git rebase origin/main
$ git stash pop

- fetch --all es para actualizar el registro de las ramas de git
- add --all agrega todo los cambios no commiteados al tracking
- stash guarda los cambios no commiteados en un baul temporario
- rebase trae los commits de la branch principal y ubica los de la branch local encima
- stash pop recupera los cambios no commiteados
